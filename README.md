# Base calculating Mode 1.12.8

Данная можификация добавляет 3 предмета: ConverterBlock, CalculatorBlock, NumberCoderBlock. Каждый из них можно полчить при помощи команды /give...

---

### Использование ConverterBlock

Пользователю необходимо ввести число в формате "{целая часть},{десятичная (если ее так можно назвать) часть}", либо в формате "{Целая часть числа}". 
Далее выбираем начальную (верхний ползунок) и конечную (нижний ползунок) системы счисления, и нажимаем кнопку *Convert* и получаем наше число в новой системе счисления.
Если введёное число содержит символы, которых нет вданной системе счисления, система вам об этом сообщит.

---

### Использование CalculatorBlock

В Calculator необходимо ввести выражение, которое необходимо посчитать. 
Ответ будет дан в наибольшей системе счисления, вычисления будут проведены по правилам арифметики: сначала умножение, потом сложение и вычитание.

Формат ввода каждого число должен быть *"{целая часть},{десятичная часть}@{cистема счисления)"* или *"{целая часть}@{cистема счисления)"*. При ошибке в вводе числа

Калькулятор поддерживает операции сложения, вычитания, умножения. Пример правильного ввода выражения: 5,5@6-10@12+4,2@11*-15@10. 
Обратите особое внимание на умножение на отрицательное число, оно записывается именно *- и никак по-другому.
Если пользователь допустит ошибку в введении какого-то числа, система сообщит об этом, однако не укажет, где именно произошла ошибка. 
При ошибки в ввдениии операций кнопка *Calculate* не будет делать ничего. Сообщения для этого не предусмотрено.

---

### Использование NumberCoderBlock

Чтобы получить прямой, обратный или дополнительный кол числа необходимо ввести его в соответствующее поле в формате *"{Целое число}@{система сисления}"* и нажать соответсвующюю кнопку.
Если пользвоатель введёт число, не существующее в данной системе или нецелое число, ему будет об этом сообщено в чате.
